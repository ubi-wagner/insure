# Landing page screenshots

Drop PNG/JPG files here with the names referenced in
`frontend/src/app/welcome/page.tsx`:

- `dashboard.png` — map + pipeline split view
- `lead.png`     — lead detail panel
- `ops.png`      — Ops Center

Recommended aspect ratio: 16:10. The frame on the landing page is
`aspect-[16/10]` and uses `object-cover object-top`, so anything wider or
taller is cropped from the bottom. ~1600x1000 is plenty.

If a file is missing the wrapper renders as a dark gradient placeholder —
the page still looks fine without screenshots.
