Frame on the landing page is `aspect-[4/3]` with `object-cover object-top`,
so taller screenshots are cropped from the bottom. To replace any of
these, drop a new PNG at the same name. Originals were captured directly
from the production app at `insure.up.railway.app`.

If a file is removed, the wrapper degrades to a dark gradient placeholder
with the alt text.
